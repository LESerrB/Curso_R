# **3. Introducción a Tidyverse**

## **Bibliografía y referencias**

### **Libros**

### ggplot2: Elegant Graphics for Data Analysis

Este libro trata puramente ggplot2 y las especificaciones. Si quieres volverte un experto no dudes en echarle un ojo.

[ggplot2: Elegant Graphics for Data Analysis](https://ggplot2-book.org/)

### R for Data Science

Del mismo autor previo, es sobre todo un tratado sobre las librerías del tidyverse y cómo se pueden utilizar para hacer ciencia de datos. Es un libro muy básico pero que consolida las bases necesarias.

[R for Data Science](https://r4ds.had.co.nz/)

### Modern Data Science with R

Es muy parecido al anterior en cuanto a objetivo, pero este es mucho más formal y con mejor tratamiento de ciertos temas. Recomiendo echarle un ojo después de leer los anteriores.

[Modern Data Science with R, 2nd edition](https://beanumber.github.io/mdsr2e/index.html)
