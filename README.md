# Curso R

Curso sobre data science con R

Notas para las instalaciones:

- Hay que instalar `R` y luego `RStudio` en orden
- Aquellos que tengan Mac deben tenerlo actualizado a la versión **OSX 10.12 o mayor**

`R` es el lenguaje de programación que estaremos usando durante el curso. 

Vamos a usar las versiones `3.6.3` (**windows**) o `3.6.2` (**Mac**).

## Instalar RStudio

`RStudio` es nuestro ambiente de programación, donde escribiremos y ejecutaremos código, y podremos ver los resultados.

Vamos a usar la versión `1.2.5033`.
